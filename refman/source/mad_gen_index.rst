########
LANGUAGE

most users interact with first: the shell, scripting model, object model,
beams, elements, sequences, tables, and MAD-X interoperability.

:doc:`mad_gen_intro`, then continue to :doc:`mad_gen_madx`,
:doc:`mad_gen_beam`, :doc:`mad_gen_sequence`, and the Part II command pages
that consume those objects.
